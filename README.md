# EcoSchool AI Power Saver

EcoSchool AI Power Saver v12.0 ULTIMATE is an enterprise-level energy management system powered by AI.

## Features

- 15 AI Agents (Full Enterprise Suite)
- MySQL Database integration
- IoT Device Control
- Smart Search System
- Per-Room Analytics
- Ultra-Intelligent Chatbot
- Auto-Learning & Self-Improvement

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tamfpt2024-stack/EcoSchool-AI-Power-Saver.git
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Setup environment variables:
   Copy `.env.example` to `.env` and fill in your credentials.

4. Run the application:
   ```bash
   python ecoschool_ultimate_v12.py
   ```
